# pattern-v2-endgame-lite-100k-mt-v0

This is the first committed learned evaluation artifact for vibe-othello-2.

Status: experimental-default.

It uses the `pattern-v2-endgame-lite` pattern set and was trained from locally
processed game transcripts published as Egaroucid training data:
"Egaroucid 7.8.0 lv.11 vs Edax 4.5.5 lv.11 game transcripts".

Teacher labels were generated by the vibe-othello exact solver. The training
target is exact after-move child final disc difference for low-empty root move
teacher positions.

Raw transcripts, normalized corpora, selected TSVs, teacher labels,
move-teacher TSVs, cache files, local reports, and logs are not redistributed.

Use this artifact explicitly with:

```sh
./build/tools/engine-cli/vibe-othello-engine-cli bestmove --moves "" --depth 4 --eval-artifact data/eval/artifacts/pattern-v2-endgame-lite-100k-mt-v0/manifest.json
```

Disable the default learned artifact with:

```sh
./build/tools/engine-cli/vibe-othello-engine-cli bestmove --moves "" --depth 4 --eval-mode static
```

This is not an official Egaroucid artifact.

This artifact is a local engineering default, not an Elo result, not a
self-play claim, and not a production-strength claim.
