This artifact was trained using locally processed game transcripts derived from
Egaroucid training data: Egaroucid 7.8.0 lv.11 vs Edax 4.5.5 lv.11 game
transcripts.

The raw Egaroucid training data and generated teacher corpus are not
redistributed in this repository.

Search labels were generated by the vibe-othello engine with the previously
committed evaluation artifact. Late-game weights retain the previous artifact's
exact-solver-trained values.

This is not an official Egaroucid artifact.
